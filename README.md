FRONTEND-WEB
Interfaz de usuario para navegadores, centrada en la experiencia del usuario que consuma o haga uso de los servicios del negocio/empresa mediante la web, integrada con la parte administrativa para la gestion de usuarios.

Entorno de desarrollo
Node version
```bash
Node v22.22.0
```
Para usar la misma version de Node ejecutar
```bash
nvm use
```
Habilitar corepack (solo una vez)
```bash
corepack enable
```
Instalar dependencias
```bash
yarn install
```
Ejecutar proyecto
```bash
yarn dev
```


Comandos git
Clonar el repositorio en la rama development 
```bash
git clone <URL_DEL_REPOSITORIO>
```
Entrar a la carpeta 
```bash
cd FRONTEND-WEB
```
Moverse a la rama development para bajar cambios
```bash
git checkout development
```
Crear y moverse a la nueva rama para trabajar
```bash
git checkout -b feature/nombre-de-tu-feature
```
